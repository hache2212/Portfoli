Task 1. Database

Create a new database named m02uf4.

Task 2. Collection

Create a new collection named movies in the m02uf4 database.

Task 3. Documents with variables

Using variables, insert the documents for the movies Fight Club and Pulp Fiction at the same time into the movies collection.

Task 4. Documents without variables

Without using variables, insert the document for Inglourious Basterds to the movies collection.

Task 5. Documents without previous collection

Insert the documents The Hobbit: An Unexpected Journey, The Hobbit: The Desolation of Smaug i The Hobbit: The Battle of the Five Armies at the same time in a collection named moviestolkien that would be created during the insertion.
